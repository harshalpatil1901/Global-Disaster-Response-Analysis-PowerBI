# 🌍 Global Disaster Response Analysis (2018–2024)

## 📌 Project Overview

Global Disaster Response Analysis (2018–2024) is an interactive Power BI dashboard designed to analyze worldwide disaster events and evaluate emergency response performance across multiple countries.

The project transforms raw disaster data into meaningful insights to understand disaster frequency, human impact, economic loss, aid distribution, and operational efficiency. The dashboard enables stakeholders to monitor trends, identify high-risk regions, and evaluate disaster response effectiveness using KPI-driven visualizations.

---

## 🎯 Business Problem

Governments and humanitarian organizations often lack a centralized analytical system to:

- Monitor disaster frequency trends
- Evaluate response efficiency
- Analyze economic and human impact
- Identify high-risk countries
- Optimize aid allocation

Without a unified dashboard, decision-makers struggle to make data-driven disaster preparedness and response strategies.

---

## 🎯 Project Objectives

- Analyze total disasters from 2018 to 2024
- Measure total casualties and economic losses
- Evaluate total aid distribution
- Assess response efficiency scores
- Identify deadliest disaster types
- Compare country-wise economic impact
- Detect yearly disaster trends
- Highlight most affected countries

---

## 📊 Key Performance Indicators (KPIs)

The dashboard tracks:

- **Total Disasters:** 50K  
- **Total Casualties:** 5M  
- **Total Economic Loss:** $253.43M  
- **Total Aid Distributed:** $12.5M  
- **Average Efficiency Score:** 87.6  
- **Countries Affected:** 20  

These KPIs provide a high-level performance summary.

## 📈 Dashboard Insights

### 1️⃣ Total Disasters by Year
- Trend analysis from 2018–2024
- Identifies peak disaster years
- Helps in forecasting future risks

### 2️⃣ Global Disaster Distribution (Map View)
- Visualizes disaster occurrences geographically
- Shows region-wise concentration
- Enables country-level filtering

### 3️⃣ Deadliest Disaster Types
- Landslide
- Tornado
- Extreme Heat
- Earthquake
- Volcanic Eruption

Analyzes total casualties by disaster type.

### 4️⃣ Country-wise Economic Loss
- Compares financial damage across countries
- Highlights top loss-generating nations
- Supports economic risk assessment

### 5️⃣ Interactive Filters
- Year filter
- Country filter
- Disaster type filter

Allows dynamic and drill-down analysis.
